# Legacy Vault - 遗产保险箱

> 你最私密的一切，安全地传承给你最爱的人。

## 愿景

一个绝对安全的数字保险箱，存放你一生中最私密、最重要的东西——密码、遗嘱、私人信件、人生故事、给后人的话。在你指定的时刻，传递给你指定的人。

## 核心理念

**零知识加密 + 智能传承**：没有任何人（包括我们）能读取你的数据。系统只负责在正确的时间，把正确的内容，交给正确的人。

## 产品定位

| 维度 | 描述 |
|------|------|
| 目标用户 | 所有成年人，尤其是有家庭责任的人 |
| 核心场景 | 存储密码/遗嘱/私信 → 设置继承人和触发条件 → 安全传承 |
| 技术壁垒 | 端到端加密 + 零知识架构 + 死亡检测协议 + 多因子继承验证 |
| 商业模式 | 免费基础存储，高级容量/多继承人/律师公证服务收费 |

## 产品特色

1. **军事级加密** - 端到端加密，零知识架构，我们也看不到你的数据
2. **灵活触发** - 死亡、特定日期、特定事件触发内容发送
3. **多类型内容** - 密码、文件、视频、录音、文字、照片
4. **分级继承** - 不同内容给不同的人，精确控制
5. **生存验证** - 智能检测用户状态，防止误触发
6. **法律效力** - 可与公证服务对接，增强法律保障

## 核心功能

### 保险箱
- 密码管理器（自动填充）
- 安全笔记
- 文件加密存储
- 媒体加密存储（照片/视频/录音）

### 遗产规划
- 设置继承人
- 设置触发条件（死亡/日期/事件）
- 内容分级分配
- 定期生存验证

### 时间胶囊
- 给未来的自己写信
- 给孩子的成长寄语
- 定时发送的消息

### 紧急访问
- 紧急联系人
- 紧急访问协议（需多人确认）
- 超时自动释放

## 平台支持

- iOS (React Native / Flutter)
- Android (React Native / Flutter)

- 浏览器扩展（密码自动填充）
