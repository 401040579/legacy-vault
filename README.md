# Legacy Vault - 遗产保险箱

> 你最珍贵的一切，安全传递给你最爱的人。

## 愿景

一个军事级加密的数字保险箱，存放你一生中最私密、最重要的东西——密码、遗嘱、私人信件、人生故事、给后人的话。在你指定的时刻，安全传递给你指定的人。

**即使是我们，也看不到你的任何数据。**

## 核心理念

**零知识加密 + 智能传承**：你的数据在你的设备上加密，只有你能解密。服务端只存储密文。系统只负责在正确的时间，把正确的内容，交给正确的人。

| 维度 | 描述 |

## 产品特色

1. **零知识加密** - 端到端加密 + Argon2id密钥派生，连我们也看不到你的数据
2. **智能传承** - 5阶段渐进式生存验证 + Shamir秘密共享，安全可靠地传递遗产
3. **多类型内容** - 密码、文件、视频、录音、文字、照片，一站式安全存储
4. **灵活触发** - 生存验证超时、指定日期、多人确认、组合条件
5. **分级继承** - 不同内容给不同的人，精确控制权限
6. **时间胶囊** - 给未来的自己或他人写信，定时安全送达
7. **紧急访问** - 紧急联系人可在等待期后访问预授权内容
8. **数据主权** - 随时导出全部数据，标准化格式，开源解密工具

## 核心功能

### 保险箱
- 密码管理器（自动填充、安全审计、泄露检测）
- 安全笔记（Markdown支持）
- 文件加密存储
- 媒体加密存储（照片/视频/录音）

### 遗产规划
- 设置守护人（继承人）+ 信任等级
- Shamir秘密共享密钥分片（t-of-n阈值）
- 触发条件引擎（多条件组合）
- 5阶段渐进式生存验证协议
- 内容分级分配

### 时间胶囊
- 给未来的自己写信
- 给孩子的成长寄语
- 定时发送的加密消息

### 紧急访问
- 紧急联系人预授权
- 等待期 + 多人确认机制
- 完整审计日志

## 安全架构

```
用户主密码 → Argon2id → 主密钥 → HKDF → 加密密钥/认证密钥
                                          ↓
                                    每条数据独立DEK
                                    AES-256-GCM加密
                                          ↓
                                    Shamir秘密共享
                                    t-of-n阈值分片
```

详见 [安全架构文档](docs/SECURITY_ARCHITECTURE.md)

## 技术栈

| 层 | 技术 |
|----|------|
| 前端 | React + TypeScript + Vite + Tailwind CSS |
| 移动端 | React Native |
| 桌面端 | Tauri (Rust) |
| 加密 | libsodium + Web Crypto API |
| 密钥派生 | Argon2id (OWASP推荐参数) |
| 数据加密 | AES-256-GCM (每条数据独立DEK) |
| 秘密共享 | Shamir SSS (GF(2^256-189)) |
| 认证 | WebAuthn / FIDO2 |
| 后端 | Rust |

## 核心承诺

1. **你的数据，你做主** - 零知识加密，我们无法也不会访问你的数据
2. **长期保障** - 数据托管信托 + 开源核心代码，即使公司消失你的数据也安全
3. **透明运营** - 核心加密代码开源，接受全球审查；年度透明度报告
4. **安全不妥协** - 不设后门，不卖数据，不投放广告
