# Legacy Vault - 开发路线图

## Phase 0：体验原型 (PWA on GitHub Pages) — 2周

### 目标
让用户体验加密存储和继承设置的基本流程

### 交付物
- GitHub Pages PWA
- 加密笔记（本地加密/解密演示）
- 基础密码管理器
- 简单继承人设置（模拟）
- 安全性可视化展示
- 品牌 Landing Page

### 技术栈
- Vite + React
- Web Crypto API (浏览器端加密)
- IndexedDB (本地加密存储)
- Service Worker + manifest.json
