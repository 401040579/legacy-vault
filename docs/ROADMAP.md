# Legacy Vault - 开发路线图

## Phase 0：体验原型 (PWA on GitHub Pages) — 2周

### 目标
让用户体验加密存储和继承设置的基本流程，验证核心概念

### 交付物
- GitHub Pages PWA
- 品牌 Landing Page（含交互式加密演示）
- 注册流程（主密码创建 + 恢复密钥生成）
- 加密笔记（本地加密/解密，Argon2id密钥派生）
- 基础密码管理器（增删改查 + 密码生成器）
- 简单继承人设置（模拟流程，不实际执行）
- 安全性可视化展示

### 技术栈
- Vite + React + TypeScript
- Web Crypto API + libsodium-wrappers (浏览器端加密)
- Argon2id 密钥派生 (m=47104, t=1, p=1)
- IndexedDB (本地加密存储)
- Service Worker + manifest.json
- Tailwind CSS

### 安全审计
- 内部代码审查：加密模块

### 同步任务
- [ ] 安全白皮书发布
- [ ] 核心加密引擎代码开源
- [ ] 隐私政策和用户协议初版

---

## Phase 1：核心加密引擎 — 6周

### 里程碑
- [ ] 完整的端到端加密系统（密钥层级：主密钥→加密密钥→DEK）
- [ ] Argon2id密钥派生 + HKDF子密钥派生
- [ ] AES-256-GCM数据加密（每条数据独立DEK）
- [ ] 多类型内容加密存储（文字/文件/媒体）
- [ ] 云端加密同步（零知识架构）
- [ ] 浏览器扩展（Chrome + Firefox，密码自动填充）
- [ ] 生物识别解锁集成（WebAuthn/FIDO2）
- [ ] 密码安全审计（弱密码/重复/泄露检测，HaveIBeenPwned k-匿名查询）
- [ ] 数据导出工具（加密或明文格式）

### 安全审计
- 第三方安全审计（加密实现+渗透测试）

---

## Phase 2：继承系统 — 8周

### 里程碑
- [ ] 继承人管理系统（邀请/验证/信任等级）
- [ ] Shamir秘密共享实现（GF(2^256-189)，可配置t-of-n阈值）
- [ ] 触发条件引擎（生存验证超时/指定日期/多人确认/组合逻辑）
- [ ] 生存验证协议（5阶段渐进式，最短31天）
- [ ] 紧急访问机制（等待期+多人确认+审计日志）
- [ ] 通知系统（邮件SendGrid + 短信Twilio，多渠道冗余）
- [ ] 继承人接收流程（身份验证 + 分片提交 + 内容解密）
- [ ] 时间胶囊系统

### 安全审计
- 加密协议形式化验证
- 漏洞赏金计划启动

### 合规
- [ ] GDPR合规审查
- [ ] CCPA合规审查
